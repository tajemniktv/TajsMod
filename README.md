## About

The plan is simple: keep expanding this into a "make the game better" toolbox ‚Äî more settings, more convenience features, more polish, hopefully without turning it into bloat.

This started as **TajsView** ‚Äî one specific "my take"/vibe for Upload Labs. But the more I worked on it, the more it became obvious I don't really want one direction. I just want to tweak everything that can be improved. So from the working title **TajsView**, it turned into simply **Taj's Mod**.

> _Also: I wanted to keep the naming vibe from my [TajsGraph](https://ficsit.app/mod/TajsGraph) mod for Satisfactory lol._

If something feels missing, annoying, or you want a specific option/toggle ‚Äî let me know! I actively read feedback for the game itself, and I'll do the same for the mod. Your suggestion might end up in the next update.

## Features

A clean bundle of QoL, visuals, and power-user tools ‚Äî with any gameplay-affecting stuff kept strictly **opt-in**.

### üéõÔ∏è General / QoL

- **Command Palette (MMB/Space)** ‚Äî Fuzzy search for commands, settings, and navigation.
- **Mod Settings Panel** ‚Äî A deeper settings menu accessible via the puzzle piece icon.
- **Toast History** ‚Äî Missed a notification? Check the log via the bell icon.
- **Mute on Focus Loss** ‚Äî Auto-mute game audio when you tab out (configurable volume).
- **Real-time Node Counter** ‚Äî Track your node usage vs the limit.
- **Smart Screenshots** ‚Äî HQ capture overrides, and resolution settings.

### üé® Visuals

- **Group Node Patterns** ‚Äî Custom patterns (Dots, Stripes, Zipzag, etc.) with adjustable opacity and colors.
- **Wire Colors** ‚Äî Customize wire colors by resource type (Power, Data, etc.).
- **Disconnected Node Highlighter** ‚Äî Visual aids (`Pulse` or `Outline` style) to find unconnected nodes.
- **Extra Glow & Bloom** ‚Äî Tweak intensity, bloom, and sensitivity of game visuals.
- **UI Opacity** ‚Äî Adjust HUD transparency.
- **Custom Boot Screen** ‚Äî A cleaner startup screen (optional).

### üîß Tools & Utilities

- **Wire Drop Menu** ‚Äî Drop a wire on empty space to instantly open a compatible node spawner.
- **Quick Navigation** ‚Äî "Jump to Group" and "Jump to Note" commands to zoom to specific areas.
- **Screenshot Selection** ‚Äî Capture only the currently selected nodes (transparent background supported).
- **Sticky Notes** ‚Äî Place colored text notes on the canvas.
- **Universal Clear Wires** ‚Äî Right-click output slots or use "Clear Wires" on a selection to clean up.
- **The Bin** ‚Äî A dedicated trash node for easy deletion.
- **6-Input Containers** ‚Äî Option to increase container inputs from 5 to 6.

### üéÆ Gameplay (Opt-in)

- **Buy Max Button** ‚Äî Smart button to buy max affordable upgrades (Lowest/Highest cost strategy).
- **Node Limit Control** ‚Äî Increase limit up to 2000 or set to Unlimited (‚àû).
- **Disable Controller Input** ‚Äî Completely block gamepad input to prevent interference.
- **Cheats Panel** ‚Äî Modify Money, Research, and Tokens (Add/Remove %, Set to 0).

> **Important:** Anything that affects gameplay is **opt-in** and clearly separated from pure QoL/visual settings.

### üêõ Debug/Troubleshooting

- **Debug Logging** ‚Äî Toggle verbose logs for troubleshooting.
- **Reset All Settings** ‚Äî One click back to defaults.
- **Debug Info** ‚Äî logs current currency values and limits.
